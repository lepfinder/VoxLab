# 2.2 麦克风音频采集原理 (I2S)

本小节将为您介绍如何在 ESP32 上通过 I2S 总线驱动 INMP441 数字麦克风采集音频，并提供 Arduino 框架下的配置代码示例。

---

## 1. 什么是 I2S 协议？
**I2S (Inter-IC Sound)** 是一种专注于 IC 芯片间音频数据传输的数字总线接口标准。它通常需要三根核心信号线：
1.  **SCK / BCLK (Bit Clock)**：串行时钟线。对应数字音频的位时钟，每一个脉冲传输 1 位数据。
2.  **WS / LRCK (Word Select)**：字选择线（左右声道选择）。用来指示当前传输的是左声道还是右声道数据，同时其频率决定了**音频的采样率**（如 16kHz）。
3.  **SD / DATA (Serial Data)**：串行数据线。传输量化后的音频数据。

---

## 2. INMP441 与 ESP32 管脚连接
INMP441 麦克风有 6 个引脚，与 ESP32 典型连接如下：

| INMP441 引脚 | 功能说明 | ESP32 引脚 (以 ESP32-S3 为例) |
| :--- | :--- | :--- |
| **VDD** | 3.3V 电源 | 3.3V |
| **GND** | 电源地 | GND |
| **L/R** | 声道选择 (接地为左声道，接 VDD 为右声道) | GND (接地) |
| **SCK** | I2S 时钟 (BCLK) | GPIO 4 |
| **WS** | I2S 字选时钟 (LRCK) | GPIO 5 |
| **SD** | I2S 数据输出 (DATA) | GPIO 6 |

---

## 3. ESP32 采集配置核心参数
为了匹配后端的接收标准，ESP32 采集音频需按以下参数进行 I2S 配置：
*   **采样率 (Sample Rate)**：`16000` (16kHz)。
*   **通道格式 (Channel Format)**：`I2S_CHANNEL_FMT_ONLY_LEFT` (单声道)。
*   **通信格式 (Communication Format)**：`I2S_COMM_FORMAT_STAND_I2S` (标准 I2S 格式)。
*   **数据位数 (Bits Per Sample)**：虽然 INMP441 硬件输出 24-bit 数据，但为了适配算法，通常配置为 **`16-bit`**，ESP32 会自动进行数据截断或对齐。

---

## 4. Arduino ESP32 采集示例代码

以下代码展示了如何使用内置的 `driver/i2s.h` 库读取音频数据：

```cpp
#include <Arduino.h>
#include <driver/i2s.h>

#define I2S_WS      5
#define I2S_SD      6
#define I2S_SCK     4
#define I2S_PORT    I2S_NUM_0

// 配置 16kHz, 单声道, 16bit PCM 采集
void setup_i2s() {
    i2s_config_t i2s_config = {
        .mode = (i2s_mode_t)(I2S_MODE_MASTER | I2S_MODE_RX),
        .sample_rate = 16000,
        .bits_per_sample = I2S_BITS_PER_SAMPLE_16BIT,
        .channel_format = I2S_CHANNEL_FMT_ONLY_LEFT,
        .communication_format = I2S_COMM_FORMAT_STAND_I2S,
        .intr_alloc_flags = ESP_INTR_FLAG_LEVEL1,
        .dma_buf_count = 8,
        .dma_buf_len = 64,
        .use_apll = false
    };

    i2s_pin_config_t pin_config = {
        .bck_io_num = I2S_SCK,
        .ws_io_num = I2S_WS,
        .data_out_num = I2S_PIN_NO_CHANGE,
        .data_in_num = I2S_SD
    };

    i2s_driver_install(I2S_PORT, &i2s_config, 0, NULL);
    i2s_set_pin(I2S_PORT, &pin_config);
    i2s_zero_dma_buffer(I2S_PORT);
}

void setup() {
    Serial.begin(115200);
    setup_i2s();
    Serial.println("I2S Mic initialized!");
}

void loop() {
    int16_t buffer[512]; // 用于存放读取的 PCM 数据
    size_t bytes_read = 0;
    
    // 从 I2S 读取数据
    esp_err_t result = i2s_read(I2S_PORT, &buffer, sizeof(buffer), &bytes_read, portMAX_DELAY);
    
    if (result == ESP_OK && bytes_read > 0) {
        // buffer 里现在存储了原始的 Int16 格式 PCM 音频，共 bytes_read 字节。
        // 下一小节我们将把这些数据打包并进行 Opus 压缩编码。
    }
}
```
通过以上代码，您已能够在 ESP32 上获得稳定的 16kHz PCM 单声道语音数据。
